Two ways for designers to ship for iOS.

Vibe coding
ToolsLovable · Bolt · Replit · v0
Pros
Fast to first prototype.
One familiar prompt bar.
Almost no setup.
Cons
Web app in a native shell.
Throttled experience with hardware and sensors.
Less code/build control and visibility.
Harder to pass Apple review.
Security and auth gaps.
Agentic building
ToolsCursor or similar, Opus model or similar
Pros
Real native Swift.
Full hardware + sensors.
Better odds of positive Apple review.
Privacy + on-device by default.
Step by step, you stay in control.
Cons
Steeper learning curve.
More to set up (IDE + Apple Dev).
Slower to the first prototype.

Four hours in your own time, for the how.
